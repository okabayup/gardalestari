# **App Name**: Garda Lestari

## Core Features:

- Digital Membership Registration: Allow users to register for membership digitally and automatically generate a KTA (Kartu Tanda Anggota/Membership Card).
- Google Authentication: Integrate with Google accounts for authentication.
- Photo/Video Feed: Display a feed of photos and videos on the home page for logged-in users, similar to Instagram.
- Member Directory: Provide a directory of members, including central administrators, regional administrators, and advisory board members.
- Blog and Events Pages: Create dedicated pages for blog posts and upcoming events.
- Directory of Programs: Feature a directory of programs with highlighted flagship programs.
- AI-Powered Personalized Benefit Recommendations: Based on user profile and usage patterns, the app will suggest relevant benefits that might interest a user. The app uses a reasoning tool to filter out benefits that do not align with user preferences.

## Style Guidelines:

- Primary color: Forest green (#347C45) reflecting the organization's association with nature.
- Background color: Light moss green (#E8F0E5), desaturated to 20% saturation for a subtle backdrop.
- Accent color: Olive green (#556B2F), an analogous color, darker and less saturated than the primary for contrast.
- Font: 'PT Sans', a humanist sans-serif font suitable for both headlines and body text.
- Mobile app-like layout with bottom navigation, full logo display, and landing page-style content on the main page.
- Use minimalist, nature-inspired icons for the bottom navigation and throughout the app.
- Subtle, active animations and glassmorphism effects with a green tint to enhance the user experience.
